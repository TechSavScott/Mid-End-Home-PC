# Home PC Build Project

## Project Overview

This project documents the planning, assembly, configuration, troubleshooting, and testing process of a custom-built desktop computer. The build was completed independently with a focus on hardware compatibility, system performance, long-term reliability, and practical hands-on experience in PC hardware and system configuration.

The project involved:

- Researching component compatibility and power requirements
- Physically assembling system hardware
- Installing and configuring Windows 10
- Updating BIOS/UEFI settings and drivers
- Diagnosing setup issues
- Stress testing and optimizing airflow for stability

---

## Hardware Specifications

| Component | Model |
|------------|--------|
| CPU | AMD Ryzen 5 5600 |
| Motherboard | MSI B550M PRO-VDH WIFI |
| Memory | 32GB DDR4 RAM |
| Graphics Card | NVIDIA GeForce RTX 4070 |
| Primary Storage | Samsung 990 PRO 1TB NVMe SSD |
| Secondary Storage | Seagate 2TB SATA HDD |
| Operating System | Windows 10 |

---

## Phase 1: Pre-Build Planning and Research

Prior to assembly, hardware research was conducted to ensure component compatibility and expected performance.

Areas researched included:

- CPU and motherboard socket compatibility
- RAM compatibility with motherboard specifications
- GPU size and system requirements
- Storage interface compatibility
- Cooling considerations
- Estimated system power requirements
- Hardware benchmark comparisons

This planning stage reduced the risk of hardware conflicts and installation issues during assembly.

---

## Phase 2: Hardware Assembly

### Step 1: Motherboard Preparation

The motherboard was prepared outside of the case to simplify installation.

Tasks completed:

- Installed AMD Ryzen 5 5600 CPU into motherboard socket
- Installed 32GB DDR4 RAM into appropriate memory slots
- Installed Samsung 990 PRO 1TB NVMe SSD into M.2 slot

---

### Step 2: Motherboard Installation

After preparing the motherboard:

- Motherboard was mounted and secured
- Connections were checked for proper seating and alignment

---

### Step 3: Graphics Card Installation

The NVIDIA GeForce RTX 4070 was installed into the motherboard PCIe slot.

Tasks completed:

- Inserted GPU into PCIe slot
- Secured card to chassis
- Verified proper connection and seating

---

### Step 4: Secondary Storage Installation

The Seagate 2TB SATA HDD was installed for additional storage capacity.

Tasks completed:

- Mounted drive
- Connected SATA data connection
- Verified storage detection

---

### Step 5: Cable Management and Airflow Optimization

Cable routing was organized to:

- Improve airflow
- Reduce cable obstruction
- Improve future maintenance accessibility
- Maintain a cleaner internal layout

---

## Phase 3: Initial Power-On and BIOS Configuration

After completing assembly, the system underwent initial startup verification.

Tasks completed:

- Verified successful system POST
- Accessed BIOS/UEFI settings
- Confirmed hardware detection:
  - CPU
  - Memory
  - Storage devices
- Reviewed system configuration settings

---

## Phase 4: Operating System Installation

Windows 10 installation was performed on the Samsung 990 PRO 1TB NVMe SSD.

Configuration process included:

- Installing Windows 10
- Creating system partitions
- Completing system setup
- Applying operating system updates

---

## Phase 5: Driver Installation and System Configuration

Following operating system installation:

Tasks completed:

- Installed motherboard drivers
- Installed graphics drivers
- Installed system updates
- Verified device recognition
- Confirmed stable operation of installed components

---

## Phase 6: Troubleshooting and Issue Resolution

During system setup and configuration, troubleshooting was performed to resolve hardware and software issues.

Issues addressed included:

- Hardware detection verification
- Driver-related configuration problems
- Display configuration adjustments
- General setup troubleshooting

Problem resolution involved systematic testing and verification of system functionality.

---

## Phase 7: Stress Testing and Validation

System testing was performed to validate performance and long-term stability.

Testing objectives:

- Verify stable system operation under load
- Monitor system responsiveness
- Validate storage functionality
- Confirm GPU and CPU performance behavior
- Ensure thermal performance remained acceptable

---

## Final Outcome

The completed build resulted in:

- Successful hardware assembly
- Stable Windows 10 installation
- Proper hardware and driver functionality
- Optimized airflow and cable management
- Reliable system performance under testing conditions
