# `Technical Implementation Summary:` 

```
The process of planning and implementing this system included verifying the Zen
3 CPU support on the B550 chipset, and installing the dual-channel memory set in
the memory slots A2 and B2 physically installing the Ryzen processor with AMD
Wraith cooling. It allows storage channels with a 1TB NVMe Samsung 990 PRO M.2
SSD and a second 2TB SATA hard drive. The operating system configurations
involved were clean installation of Windows 10 using boot media on a 50gb USB
stick, setting up the custom UEFI configuration, and an installation of the
chipset and hardware drivers. The stability and physical design limits of the
system were then tested with 3DMark benchmark stress/performance testing.
```
